# mynumpy
Bu resposi’de NumPy’ın içindeki kendi zeros fonksiyonumuzu yazıyoruz. Amacımız da mantığını anlamak.

1– İlk adım: Kullanıcıdan satır ve sütun sayısını girmesini istiyoruz.  
2– Eğer sadece satır sayısını girmişse, if ile ne girdiğini teyit edip, satır sayısını sütuna eşitleyerek kare matris oluşturacağımızı anlıyoruz.  
3– For döngülerine geçerken ilk olarak ana matrisimizi, yani zero matrisimizi oluşturuyoruz.  
4– For içine açtığımız diğer for ile ana matrisin içindeki satır matrislerini oluşturuyoruz.  
5– Adım adım satır matrisine 0’ları ekliyoruz.  
6– Satır matrisini ana matrise ekliyoruz ve döngü bu şekilde devam ediyor.  
7– En son return ediyoruz.
